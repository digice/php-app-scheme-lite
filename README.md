# php-app-scheme-lite
Application structure for a default PHP Application (SQLite Version)
`
<#appname#>/
├── bin/
│   ├── main.php
│   ├── setup.php
│   ├── test.php
├── dep/
│   ├── <#dependency#>
│   └── <#dependency#>
├── etc/
│   ├── config.php
│   └── users.csv
├── lib/
│   ├── .abc.php
│   └── <#class#>.php
└── var/
    ├── db
    ├── inc
    ├── tmp
    └── www
`
